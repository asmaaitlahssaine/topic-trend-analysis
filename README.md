# topic-trend-analysis

Automatic discovery of customer-complaint topics, validated for coherence, and
organized into a navigable hierarchy for trend analysis. 

## Key results

- Topics discovered automatically, with no predefined categories
- Coherence validated without labels (ProxAnn-inspired score)
- Topics organized into an interactive, expandable hierarchy

## Files

- `notebook.ipynb`: full pipeline with explanations
- `topic_treegraph.html`: interactive hierarchical visualization

## Viewing the interactive tree

The hierarchy is in `topic_treegraph.html`. To view it interactively:

1. Download the file and open it in a browser

## Value

The same hierarchy serves both a high-level overview and a detailed drill-down,
and provides the foundation for analysing complaint trends at any level of depth.
